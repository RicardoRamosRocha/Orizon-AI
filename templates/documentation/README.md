# Template Veltis Documentation

Template padrão de documentação para projetos desenvolvidos no ecossistema Veltis.

## Estrutura

- 01-visao-produto.md
- 02-requisitos.md
- 03-backlog.md
- 04-roadmap.md
- 05-modelagem-banco.md
- 06-arquitetura.md

- adr/

## Objetivo

Padronizar a documentação de projetos SaaS, ERP, CRM, EAD, APIs e sistemas administrativos.

## Uso

Copiar esta pasta para um novo projeto e adaptar os documentos conforme o domínio do sistema.
