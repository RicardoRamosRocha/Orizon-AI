# Veltis AI

Plataforma SaaS para gerenciamento de clientes, planos, créditos e acesso centralizado a múltiplos provedores de Inteligência Artificial.

## Tecnologias

- ASP.NET Core 8
- Entity Framework Core
- PostgreSQL
- Web API
- MVC
- Bootstrap 5
- Flutter futuramente

## Módulos Planejados

- Administração
- Clientes
- Planos
- Créditos
- Assinaturas
- Provedores de IA
- Modelos de IA
- Chat
- Controle de consumo
- Financeiro
- Relatórios
